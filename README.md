# Calculadora Comercial

Este projeto é uma **calculadora** desenvolvida para facilitar o cálculo de preços, janelas de pagamento, taxas e impostos, com o objetivo de **auxiliar vendedores e gerentes** na criação de propostas comerciais precisas e otimizadas.

## Funcionalidades

- **Cálculo de Preços**: Insira o valor de produtos e serviços para calcular o preço final.
- **Janelas de Pagamento**: Ajuste os prazos e veja como isso impacta o fluxo financeiro.
- **Taxas**: Calcule automaticamente as taxas aplicáveis em cada transação.
- **Impostos**: Integração para cálculo de impostos comuns sobre vendas.

## Objetivo

A ferramenta foi projetada para **uso pessoal** e para ajudar colegas a realizar cálculos financeiros de maneira mais eficiente e precisa. Além disso, serve como um guia essencial para vendedores e gerentes ao elaborar propostas comerciais, economizando tempo e minimizando erros.

## Tecnologias Utilizadas

- **HTML5**: Estrutura do site.
- **CSS3**: Estilos e design responsivo.
- **JavaScript**: Lógica de cálculo e interatividade da página.

## Como Usar

1. Insira o preço base dos produtos ou serviços.
2. Escolha o prazo da janela de pagamento.
3. Insira as taxas aplicáveis e veja o valor final.
4. Veja o cálculo automático dos impostos.
5. Use os resultados para otimizar as suas propostas comerciais.
